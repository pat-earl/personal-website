# personal-website
## Patrick's Personal Website

## Why? 
I wanted to challenge myself to design a website using CSS without a framework and had a domain that was sitting dormant.

## How?
Well I'm using Python + Flask. It's hosted on a VPS with NGINX and WSGI. 

"It's just a static website, why does he need Python/Flask to host it?" Yes, that's very observant of you, but I'm also into system administration. Apache2/PHP stack has been done so many times and I've wanted to explore setting up a different web stack. Besides I :heart: Python and possibly add something that takes advantage of the stack.
